HTML, CSS et JavaScript facile

  • JavaScript
  • CSS

En général (comme pour la plupart des domaines), si vous voulez commencer à apprendre le language informatique, il faut un bonne raison, ça peut-être par exemple :

  • Développer un site web (personnel, ou professionnel) ;
  • Développer un logiciel ;
  • Devenir développeur...

Bref pour toute les raisons que vous voulez sauf une ! Une seule qui est vraiment interdite, c'est pour dire à tout le monde je suis plus intelligent qua vous ou pour séduire plus facielement... Car si c'est pour ça vous allez très vite oublier les méthodes, et autres techniques d'pprentissages. Et vous allez pas trop vous améliorer. Car moi, une fois que j'ai appris les bases, j'ai continué à m'amélioré dans le language car j'ai réfléchie dans ce que je pourrai faire de plus...

Il faut aussi savoir à quoi sert le language HTML, CSS, et JS (JavaScript) :

  • HTML : ou Hypertext Markup Language est le language de base, celui avec lequel l'enemble des sites internets sont fait, c'est celui qui permet d'afficher du texte ;
  • CSS : ou Feuilles de styles en cascades est un language très simple à apprendre. Il permet entre autre de faire ce que j'appelle du maquillage au site. Par exemple, un site sans CSS ressemble à du traitement de texte simple sur Word (avec du texte, des images, des listes à puces, des tableaux simples...), mais le CSS, va faire ressembler votre site à ce site, que vous visitez ;
  • JavaScript : comparé aux deux précédents qui sont presques obligatoires, celui-ci on peut s'en dispenser. Mais il permet d'aller du simple défilement de page comme sur Wix, au achat en ligne, ou encore à la création de compte sur un site. Le JavaScript permet aussi d'afficher une boîte de dialogue en haut d'une page.

Les deux premiers languages sont plus simple que ce qu'on croit à apprendre, le dernier aussi, même s'il est déjà un peu pus complexe, mais plus simple que ce qu'on croit.

Il est important de noter que cette formation ne vous permettra pas de vous inventer développeur web, mais elle vous permettra d'avoir des grandes bases en informatique.
