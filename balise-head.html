La balises head, est la balise non-obligatoire, mais qui va changer la donne du site. La balise head, est composé de diverses balises à l'intérieur, qui permettent comme je l'avais dit dans mon précédent cours, permet le référencement SEO. Certain me diront n'importe quoi (les développeurs), alors que si. Parce que oui, elle permet la gestion du sites, le maquillage..., mais il faut savoir que le CSS, fait partit du référencement SEO. Bref... Voyons comment s'en servir.

Balises meta

Je n'ai jamais compris l'utilité, mais dans la balise head insérer ceci :

                
                    <meta charset="utf-8">

Balises title

Cette balise est obligatoire pour le SEO, et devrait être obligatoire tout court.

Sur cette image, pour que le texte About discards s'affiche, il faut tou d'abord, ajouter les balies title.

Exemple :

                
                    <title>About discards</title>

Balises link

Bon pour le moment nous n'en avons pas besoin, mais par la suite, elle nous sera très utile. Cette balise unique permet de faire plein de chose :

  • Créer une redirection sur le site s'il est consulté sur un mobile ;
  • Créer une redirection sur le site s'il est consulté depuis une langue que celui qui est utilisé sur le site ;
  • Déclarer un fichier CSS.

Pour le moment, nous n'allons pas nous intéresser à ça, mais bientôt dans les cours de CSS, on va voir ça.

Balises script

Là encore, ça nous est inutile pour le moment. Mais voici deux exemples d'utilisations :

Création d'un script sur la page.

                
                    <script>
                    <!-- script -->
                    </script>

Déclaration d'un script :

                
                    <script src="chemin_du_script.js" type="text/javascript">
